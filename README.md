# ⚡ Energy Tracker

PWA para acompanhar água, calorias, proteína, carboidrato e gordura diários.

## Como usar

### Opção 1 — GitHub Pages (recomendado)

1. Crie um repositório no GitHub (pode ser público ou privado)
2. Faça upload dos 3 arquivos: `index.html`, `sw.js`, `README.md`
3. Vá em **Settings → Pages → Source: main / root**
4. Aguarde 1-2 minutos → seu app estará em `https://SEU-USUARIO.github.io/NOME-DO-REPO`

### Instalar no celular

**Android (Chrome):**
Um banner aparece automaticamente → toque em **Instalar**

**iPhone (Safari):**
Toque em **Compartilhar → Adicionar à Tela de Início**

## Funcionalidades

- Rastreamento de: 💧 Água · 🔥 Calorias · 🥩 Proteína · 🍞 Carbo · 🥑 Gordura
- Metas editáveis
- Barra de progresso verde/vermelho
- Dados salvos localmente (localStorage)
- Funciona offline (Service Worker)
- Tela cheia automática
- Splash screen com animação
